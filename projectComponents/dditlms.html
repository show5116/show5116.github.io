회원가입
정규식 처리를 한 회원가입 페이지 입니다.
로그인
Spring Security를 이용하여 로그인 권한을 잡아주었습니다.
remember-me 기능을 이용해 브라우져가 꺼져도 로그인 유지기능을 추가하였습니다.
로그인 실패가 5회이상 누적될시 잠금기능을 추가하였습니다.
메인페이지
위젯을 커스텀으로 추가나 삭제가 가능하게 구현하였습니다.
drag & drop으로 위젯의 위치를 변경할 수 있게 구현하였습니다.
채팅
원하는 유저들과 채팅방을 만들고,
websocket을 이용해 실시간 채팅을 구현하였습니다.
메일
@ddit.site 도메인을 구입하여, 고유 메일 도메인을 만들고
메일 서버를 구축하여 메일 송, 수신을 구현하였습니다.
전자 결재1
현재 결재 진행상황을 볼 수 있고,
결재 승인이나 반려를 할 수 있는 페이지 입니다.
전자 결재2
결재선을 추가하여 결재 기안을 할 수 있도록 구현한 페이지입니다.
전자 결재3
기안서를 자동으로 생성해주어 기안할 수 있도록 구현하였습니다.
클라우드
NCP(Naver Cloud Platform)을 이용하여 개인 파일 서버를 구현하였습니다.
캘린더
Calendar Api를 이용하여 캘린더 등록,수정,삭제를 구현하였습니다.
coolsms와 Spring Schedular를 이용해 일정 시작 전에 문자 알람이 가는 기능을 구현하였습니다.
과목 등록
교과 과목을 등록 및 삭제할 수 있는 페이지 입니다.
excel파일을 이용하여, 현재 교과 과목을 불러올 수 있고, 또한 새로이 등록 가능합니다.
교과 과정
앞서 등록된 과목을 특정 과의 교과 과정으로 등록할 수 있습니다.
excel파일을 이용하여, 현재 교과 과목을 불러올 수 있고, 또한 새로이 등록 가능합니다.
